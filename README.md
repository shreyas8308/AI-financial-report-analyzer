
# AI Financial Report Analyzer

A Retrieval-Augmented Generation (RAG) application that allows users to upload annual reports and ask natural language questions.

## Features

- PDF report upload
- Text extraction using PyPDF
- Semantic search using FAISS
- Local embeddings using Sentence Transformers
- Gemini-powered financial analysis
- Streamlit web interface

## Tech Stack

- Python
- Streamlit
- Gemini API
- FAISS
- Sentence Transformers
- LangChain

## Run Locally

pip install -r requirements.txt

streamlit run app.py
